Imperio Admin is a multi purpose template that can be used for any backend applications.

Features
--------
 - Responsive layout
 - Bootstrap 2.0.3
 - jQuery 1.7.2
 - jQuery Form Validation
 - jQuery Flot
 - jQuery DataTables
 - jQuery FullCallendar
 - Bootstrap Datepicker
 - Bootstrap Colorpicker
 - Additional Glyphicons

Compatible browsers
-------------------
 - Firefox 4+ 
 - Latest Chrome
 - Latest Safari
 - Opera 11
 - Internet Explorer 8+
